# 林间日记 · 双相心境记录

一个纯前端的双相情感障碍自我观察工具：记录心情、睡眠、服药、经期，自动总结规律并生成可打印的就诊报告。所有数据只保存在使用者自己设备的浏览器里，不上传任何服务器。

## 把这个仓库变成网址（只需一次）

1. 打开仓库的 **Settings → Pages**
2. Source 选 **Deploy from a branch**，Branch 选 **main**、目录选 **/(root)**，点 **Save**
3. 等 1–2 分钟刷新此页，顶部会出现你的网址：`https://<用户名>.github.io/<仓库名>/`
4. 手机打开网址即可安装：安卓 Chrome 菜单里点「安装应用」，iPhone 用 Safari 打开后点分享按钮里的「添加到主屏幕」

## 以后怎么更新版本

仓库页面点 **Add file → Upload files**，把新的 `index.html` 拖进来覆盖，Commit。约一分钟生效；已安装的用户刷新两次内会自动拿到新版，且不影响他们已保存的记录。

## 数据与隐私（请分享给使用者）

每位使用者的数据只存在自己设备上，站长也看不到；换设备或清除浏览器数据会丢失记录，请定期用「报告」页导出 .txt 备份。本应用是自我观察工具，不能替代专业诊疗。

> ⚠️ 一旦有真实用户开始记录，请不要更换网址/域名——记录是跟着域名走的，换了域名用户就看不到自己的历史数据了。想绑自定义域名请在正式分享前决定。
