# 林间日记 🌲

双相情感障碍自我观察工具。像看四季流过森林一样，记录每一天的起伏。

**在线使用**：<https://maoo1103.github.io/forest-diary/>

手机安装为 App：安卓 Chrome 打开网址 → 菜单 → 「安装应用」；iPhone Safari 打开网址 → 分享 → 「添加到主屏幕」。安装后可完全离线使用。

## 功能

记录：心情（双相临床常用的 −3 ～ +3 七级量表）、睡眠时长与质量、食欲、精力、逐药打勾的服药记录（支持自定义药名/剂量/服法）、影响因素标签（咖啡、饮酒、压力、冲突等，可自定义）、天气、月经周期（自动推算周期天数）与自由备注，可补记过去的日子。

总结：按心情上色的月历、整月「情绪之河」曲线、记录天数/心情均值/平均睡眠/服药依从率统计。

规律：自动分析睡眠、服药、经期、经前、天气、各影响因素与心情的关联；基于近 7 天走势外推未来 3 天；识别「心情走高 + 睡眠骤减」等躁狂/轻躁前兆并提示尽快联系医生。

报告：一键生成完整就诊报告（总体概况、心情分布、显著波动日、规律、每日明细），可下载 .txt 带去门诊；另有数据文件导出/导入（.json），用于换手机搬家与个人备份。

首次打开展示 45 天示例数据供体验，一键清空后开始真实记录。

## 隐私与数据

所有记录只保存在使用者自己设备的浏览器中（localStorage），不会上传到任何服务器，站点维护者也无法看到任何人的数据。这意味着：不同设备之间不互通；清除浏览器数据会同时清除记录。请定期使用「报告」页导出 .txt 备份。

## 免责声明

林间日记是自我观察的辅助工具，不构成医学建议，不能替代专业诊疗。「规律」页的预测仅为近期趋势的简单外推，不是医学预测。如处于急性期或出现危机征兆，请立即联系你的医生或当地危机干预资源。

## 维护者如何更新（给自己的备忘）

1. 获取新版 `index.html`（或其他文件）
2. 仓库页面 → `Add file` → `Upload files` → 拖入新文件覆盖 → `Commit changes`
3. 1–2 分钟后自动上线；使用者下次联网打开会自动拿到新版本，已保存的数据不受影响
4. 改坏了可在 commit 历史中回滚

## 技术说明

单文件应用：React 19 已完整内嵌于 `index.html`，无任何外部依赖，无构建步骤。`sw.js` 提供离线缓存（缓存优先 + 后台更新），`manifest.webmanifest` 提供安装能力。所有路径均为相对路径，可部署在任意子目录下。
